The five canonical paths from Microsoft Cloud Adoption Framework. Each workload gets one decision, defended by a one-page brief: business value, risk, cost, time, reversibility. Decisions are revisited only when one of the five inputs materially changes.Os cinco caminhos canônicos do Microsoft Cloud Adoption Framework. Decisões são revisitadas só quando uma das cinco entradas muda materialmente.Los cinco caminos canónicos del Microsoft Cloud Adoption Framework.
