Papers, palestras e material-fonte consultados na produção deste playbook.

4 min de leitura 3 seções

Fontes primárias

O playbook se apoia em dezenas de engajamentos com times Fortune 500, de 2024 até Q1 2026, mais benchmarks públicos e papers peer-reviewed. As referências abaixo são as citadas neste site. A bibliografia completa do playbook é cerca de 3× maior.

Benchmarks

  • FeatureBench, ICLR 2026. O gap 7× entre benchmarks de bug-fix e feature em agentes de coding comerciais.
  • SWE-bench Verified, OpenAI, 2024. Baseline de bug-fix que tornou a afirmação do FeatureBench contrastável.
  • MLOps Community State of Deployment, 2025. Os 47% de orgs fazendo deploy poucas vezes por ano.

Arquitetura

  • OpenTelemetry Semantic Conventions for GenAI, CNCF, fim de 2025. Fonte do formato de span gen_ai.* usado no Capítulo 01.
  • MCP (Model Context Protocol), Anthropic, 2024. O formato de fio entre agentes e tools.
  • Kubernetes Scheduler Framework, SIG-Scheduling, em andamento. Background para o padrão workload-descriptor no Capítulo 01.

Contexto e intenção

  • ACE, Augmented Context Engineering, NeurIPS 2025. O resultado de 32% de redução de tokens para contexto curado.
  • EARS, Easy Approach to Requirements Syntax, Mavin et al, 2009. A gramática reusada para specs de agente no Capítulo 04.
  • Spec-Driven Development for LLM Agents, Vishnyakova, 2026. O framing de specs como superfície de controle.

Governança

  • Policy as Code patterns, CNCF TAG-Security, 2025. Background para o modelo constituição-como-enforcement.
  • Os Quatro Pilares do Controle de Plataforma, framework interno, consolidado de três engajamentos.

Entrevistas

Quando praticantes são citados sem atribuição, é porque o engajamento está sob NDA. Quando números são citados, são reproduzíveis de benchmarks públicos ou de case studies publicados; a organização específica é anonimizada, mas o número é real.

Sobre a prática de citação

Números neste site são arredondados a duas casas significativas quando a amostra subjacente é pequena. Valores em dólar assumem USD de 2026. Onde uma fonte foi atualizada entre drafts, a versão mais recente é citada.
