1. The factO fatoEl hecho

On June 1, 2026, GitHub Copilot stops charging per request. Premium Request Units retire. GitHub AI Credits take over. 1 AI Credit equals US$ 0.01. Input, output and cached tokens are accounted for at the published API rate of each model.

The difference is structural. Under PRUs, cost was per user intent. Under tokens, cost is per work performed. A 4-hour agentic session refactoring 50 files is genuinely 100x more expensive than a single chat question. The billing model now reflects this.

2. The thesisA teseLa tesis
