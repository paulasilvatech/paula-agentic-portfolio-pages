1. OverviewVisão geralResumen

The modern coding assistant is a composition of six concepts: agents, skills, plugins, MCP servers, hooks and a configuration hierarchy. Each concept has its own file format, lifecycle and contract with the others. Understanding their boundaries is what separates a workspace that scales from one that breaks at the second use case.

O assistente de código moderno é uma composição de seis conceitos: agentes, skills, plugins, servidores MCP, hooks e uma hierarquia de configuração. Cada conceito tem formato de arquivo, ciclo de vida e contrato com os demais. Entender as fronteiras é o que separa um workspace que escala de um que quebra no segundo caso de uso.

2. The thesisA teseLa tesis

Agents pick the path. Skills pack the knowledge. Plugins reach into the world. MCP servers expose typed tools. Hooks keep the rails. The configuration hierarchy decides who wins when they disagree. Compose all six on purpose, and the workspace scales from one developer to the whole org.
Agentes escolhem o caminho. Skills empacotam o conhecimento. Plugins alcançam o mundo. Servidores MCP expõem ferramentas tipadas. Hooks mantêm os trilhos. A hierarquia de configuração decide quem vence quando há discordância. Componha os seis com intenção, e o workspace escala de um dev para a organização inteira.
Los agentes eligen el camino. Las skills empaquetan el conocimiento. Los plugins alcanzan el mundo. Los servidores MCP exponen herramientas tipadas. Los hooks mantienen los rieles. La jerarquía de configuración decide quién gana cuando hay desacuerdo. Componlos los seis a propósito, y el workspace escala de un dev a la organización entera.

3. The numbers that justify the effortOs números que justificam o esforçoLos números que justifican el esfuerzo

  • 4 questions, 3 outcomes. The decision tree resolves any new task in under a minute.
  • 6 prompt-integration layers. System, project, user, session, skill, tool. Order matters.
  • 3 configuration tiers. Enterprise, project, user. Last-write-wins with explicit precedence.
  • 6 hook events. PreToolUse, PostToolUse, UserPromptSubmit, Stop, Notification, SubagentStop.
  • 2 deck formats. Skills & Agents (60 min) and GitHub Copilot in VS Code (75 min).
